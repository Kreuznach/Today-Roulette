# MVP 기능 명세

## 핵심 기능 (Must Have)

### 1. 카테고리 선택

- 5개 카테고리 중 1개 선택: **에너지**, **소비**, **식사**, **소셜**, **미션**
- 각 카테고리는 고유 컬러, 아이콘, 설명 제공
- 이미 오늘 카테고리를 선택한 경우 → 이전 선택 유지 표시

### 2. 룰렛 스핀

- SVG 기반 12칸 룰렛 휠
- "룰렛 돌리기" 버튼 클릭 → 2.8초 회전 애니메이션 → 결과 랜딩
- 하루 1회 제한 (KST 기준 자정 초기화)
- 결과는 선택된 카테고리의 12개 아이템 중 무작위 1개

### 3. 1차 결과 확인

- 결과 카드: 제목, 요약, 행운 포인트, 주의 포인트, 추천 행동
- 두 가지 선택지 제공:
  - **오늘 결과 확정** → 최종 결과 화면으로 이동
  - **광고 시청 후 재추첨** → 광고 보고 같은 카테고리 내 재추첨

### 4. 광고 재추첨

- 하루 1회 재추첨 기회
- 광고 시청 완료 시에만 재추첨 허용
- 광고 닫기(dismissed) 또는 실패 시 원래 결과 유지

### 5. 최종 결과 저장

- 확정된 결과를 localStorage에 저장
- 재추첨 여부 표시
- "내일 자정 이후 다시 플레이 가능" 안내

### 6. 기록 조회

- 최근 7일 플레이 기록 목록
- 클릭 시 상세 결과 바텀 시트로 표시

## 비기능 요건

| 항목 | 기준 |
|------|------|
| 첫 화면 로딩 | < 2초 |
| JS 번들 크기 | vendor 청크 < 200KB gzip |
| 오프라인 지원 | 없음 (데이터 모두 번들 내 포함) |
| 최소 지원 | iOS 15 / Android 10 (Toss WebView 기준) |
